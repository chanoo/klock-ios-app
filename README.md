# 클라크 iOS 앱

## Development Environment

* Xcode Cloud (CI/CD)
* Swift Lint (Code Style)
* Alamofire (Network)
* Swinject (Dependency Injection)
* Combine (Reactive Programing)
* MVVM (Architecture)
* CoreData (Local Storage)
